## Сортировка вставкой (Insertion Sort)

Сортировка вставкой (Insertion Sort) - это простой алгоритм сортировки, который сортирует массив постепенным построением отсортированной части массива. Алгоритм подходит для небольших массивов или массивов с большим количеством упорядоченных элементов. Он основан на принципе вставки элемента в правильное место в отсортированной части массива.

Пример реализации сортировки вставкой на языке Java:

```java
public class InsertionSort {
    public static void main(String[] args) {
        int[] array = {5, 2, 9, 1, 5, 6};
        System.out.println("Исходный массив:");
        printArray(array);

        insertionSort(array);

        System.out.println("\nОтсортированный массив:");
        printArray(array);
    }

    public static void insertionSort(int[] array) {
        int n = array.length;
        for (int i = 1; i < n; i++) {
            int key = array[i];
            int j = i - 1;

            // Перемещаем элементы массива, которые больше key, на одну позицию вперед
            while (j >= 0 && array[j] > key) {
                array[j + 1] = array[j];
                j = j - 1;
            }
            array[j + 1] = key;
        }
    }

    public static void printArray(int[] array) {
        for (int i = 0; i < array.length; i++) {
            System.out.print(array[i] + " ");
        }
    }
}
```

Этот код демонстрирует простую сортировку вставкой. Основная идея заключается в том, чтобы проходить по массиву, начиная с элемента под индексом 1, и вставлять текущий элемент на правильное место в отсортированной части массива. Внутренний цикл while выполняет сравнения и сдвигает элементы, пока не находится правильная позиция для вставки текущего элемента.

Пример вывода программы:

```
Исходный массив:
5 2 9 1 5 6 

Отсортированный массив:
1 2 5 5 6 9 
```
## Рекурсия

Рекурсия - это прием программирования, при котором функция вызывает сама себя, либо непосредственно, либо через цепочку других вызовов. Рекурсивные функции обычно имеют условие выхода, чтобы предотвратить бесконечное выполнение.

Пример рекурсивной функции на языке Java, реализующей возведение числа в степень:

```java
public class RecursionExample {
    public static void main(String[] args) {
        int base = 2;
        int exponent = 5;

        int result = power(base, exponent);

        System.out.println(base + " в степени " + exponent + " = " + result);
    }

    public static int power(int base, int exponent) {
        // Условие выхода из рекурсии
        if (exponent == 0) {
            return 1;
        } else {
            // Рекурсивный вызов функции
            return base * power(base, exponent - 1);
        }
    }
}
```

В этом примере функция `power` рекурсивно вызывает саму себя до тех пор, пока `exponent` не достигнет 0. Когда `exponent` равен 0, функция возвращает 1, что является условием выхода из рекурсии. В противном случае функция умножает `base` на результат рекурсивного вызова с уменьшением `exponent` на 1.

Пример вывода программы:

```
2 в степени 5 = 32
```

Однако стоит отметить, что рекурсивные решения могут привести к переполнению стека вызовов для больших значений `exponent`. В некоторых случаях более эффективным может быть использование итеративных подходов.
